# Nest.JS Calculator

<br>

### How to use it?

**Go to "http://localhost:7200/calculator/ " at the end you should wirte the operation with numbers you would like to complete**

<br>

#### For Example:
**10.294 * 203 -> http://localhost:7200/calculator/multiply?numberOne=10.94&numberTwo=203**

<br>

## Operation names
##### add
##### subtract
##### multiply
##### divide
##### power
##### root
